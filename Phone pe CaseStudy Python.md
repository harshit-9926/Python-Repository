# 📊 PhonePe Transaction Data Analysis (Python Project)

## 📌 Project Overview

This project focuses on analyzing PhonePe transaction data using Python to uncover meaningful insights related to transaction trends, user behavior, and regional performance. The analysis helps in understanding how digital payments are growing across different states and time periods.

## 📂 Dataset Description

The dataset includes information related to PhonePe transactions such as:

* **State** – Region where transactions occurred
* **Year / Quarter** – Time period of transactions
* **Registered user** – Total Registered user
* **Transaction Count** – Number of transactions
* **Transaction Amount** – Total transaction value
* **Average Transaction Amount** – Avg. transaction value
